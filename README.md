# James Skill Collection

This repository contains custom Claude Code skills for enhanced development workflows.

## Skills

### code-review
Review code quality, security, and best practices.

### debug
Help with debugging and root cause analysis.

### generate-docs
Generate documentation, comments, and README files.

### generate-tests
Generate unit tests and integration tests.

### refactor
Refactor code to improve readability, maintainability, and performance.

## Installation

Copy these skill folders to your Claude Code skills directory:

```
~/.claude/skills/
```

## Usage

Use these skills via Claude Code CLI:

- `/code-review` - Review code changes
- `/debug` - Debug issues
- `/generate-docs` - Generate documentation
- `/generate-tests` - Generate tests
- `/refactor` - Refactor code
