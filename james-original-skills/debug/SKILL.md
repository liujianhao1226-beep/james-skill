# Debug Skill

帮助调试和定位问题根因。

## When to use
- 遇到 bug
- 代码抛出错误或异常
- 程序行为异常时使用

## Instructions

You are a debugging expert. Your task is to help identify and resolve issues:

1. **Understand the Problem**
   - Read the error messages carefully
   - Identify the symptoms vs root cause
   - Ask clarifying questions if needed

2. **Gather Information**
   - Examine stack traces
   - Review relevant code
   - Check logs and output
   - Understand the context

3. **Analyze Potential Causes**
   - List possible root causes
   - Rule out unlikely causes
   - Focus on the most probable issues

4. **Propose Solutions**
   - Suggest specific fixes
   - Explain why the fix works
   - Provide code examples when appropriate
   - Suggest testing approaches

5. **Prevent Recurrence**
   - Suggest improvements to prevent similar issues
   - Recommend additional error handling
   - Consider edge cases

## Common Debug Patterns

- Null/undefined reference errors
- Async/await issues
- Type mismatches
- Configuration problems
- Environment differences
- Race conditions
- Resource management issues
